## 📌 Project Overview

This project is an audit of an open-source software system. It covers:

* Origin and philosophy of Python
* License and ethical considerations
* Linux footprint and working
* Ecosystem and dependencies
* Comparison with proprietary software

Additionally, 5 shell scripts are created to demonstrate Linux and automation concepts.

## ⚙️ How to Run Scripts

1. Open terminal in Linux
2. Navigate to project folder
3. Give permission:

   ```bash
   chmod +x script_name.sh
   ```
4. Run script:

   ```bash
   ./script_name.sh
   ```
